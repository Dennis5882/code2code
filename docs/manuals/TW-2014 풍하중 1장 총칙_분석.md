# TW-2014 풍하중 1장 총칙 분석

## 1. 結構力學核心邏輯
本章為總則，定義了風力計算的適用範圍、符號及專有名詞。核心邏輯是為後續章節（第二章至第六章）提供基礎框架，明確哪些結構物需進行風力計算，以及如何定義關鍵參數。

- **Clause 1.1 適用範圍**：適用於封閉式、部分封閉式與開放式建築物及地上獨立結構物之風力計算。
- **Clause 1.2 符號說明**：列出各符號之意義與單位。
- **Clause 1.3 專有名詞定義**：定義基本設計風速、主要風力抵抗系統、局部構件及外部被覆物、開放式/部分封閉式/封閉式建築物、開口、設計風壓、設計風力、普通/柔性建築物、用途係數、有效受風面積、特徵面積等。

## 2. 主要公式與標準
本章無具體公式，但定義了後續計算所需的參數與分類標準。

## 4. 批判性審查與疑慮
- 本章僅為總則，無具體計算，但需注意專有名詞定義可能與其他規範不一致。
- 開口定義對建築物分類（封閉式/部分封閉式）影響重大，需明確開口面積比例。
- 有效受風面積與特徵面積的定義可能影響局部構件風壓計算。
