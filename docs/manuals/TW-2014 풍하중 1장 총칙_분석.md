# TW-2014 풍하중 1장 총칙 분석

## 1. 結構力學核心邏輯
本章為總則，定義了風力計算的適用範圍、符號及專有名詞。核心邏輯是為後續章節（第二章至第六章）提供基礎框架，明確何種結構物需進行風力計算，以及計算中使用的符號和術語。

## 2. 主要公式與標準
無具體公式，但定義了關鍵參數如基本設計風速、設計風壓、設計風力等。

## 3. 數據流向圖
```
輸入：結構類型（封閉式/部分封閉式/開放式）
       → 判斷適用範圍（1.1節）
       → 引用符號定義（1.2節）
       → 確認專有名詞（1.3節）
       → 輸出：後續章節計算所需之參數與分類
```

## 4. 批判性審查與疑慮
- 適用範圍明確，但未定義「地上獨立結構物」的具體邊界條件。
- 專有名詞定義中，「有效受風面積」與「特徵面積」的區分需依賴後續章節的具體計算方法。
- 無邊界情況（如極端地形或非標準開口）的處理規則。
