- 風速垂直分布指數律（2.5式）
- 地況分類（A、B、C）及臺灣地區基本設計風速分區
- 用途係數（2.5節）
- 風速壓計算（2.6節）
- 陣風反應因子（2.7節）
- 風壓係數與風力係數（2.8節）
- 內風壓係數（2.9節）
- 橫風向風力（2.10節）
- 扭矩（2.11節）
- 風力組合（2.12節）
- 低層建築物設計風力簡化計算（2.13節）

## 2. 主要公式與標準
- 設計風壓：
  - 封閉式/部分封閉式：p = qGCp - qi(GCpi) (2.1)
  - 開放式：p = qGf Cp - qi(GCpi) (2.2)
  - 屋頂突出物：pp = qp(GCpn) (2.3)
  - 主要風力抵抗系統：F = q(z_Ac) G Cf Ac (2.4)
- 風速垂直分布：Vz = V10 (z/10)^α (2.5)
- 風速壓：q(z) = 0.06 K(z) Kzt [I V10(C)]^2 (2.6)
  - K(z) = 2.774 (z/z_g)^(2α) for z > 5m; = 2.774 (5/z_g)^(2α) for z ≤ 5m (2.7)
  - Kzt = (1 + K1 K2 K3)^2 (2.8)
- 陣風反應因子：
  - 一般建築物：G = 1.927 (1 + 1.7 I_z √(g_Q^2 Q + g_V^2)) / (1 + 1.7 g_V I_z) (2.9)
  - 柔性建築物：G_f = 1.927 (1 + 1.7 I_z √(g_Q^2 Q + g_R^2 R)) / (1 + 1.7 g_V I_z) (2.13)
- 橫風向風力：
  - 矩形建築物：WLz = 0.87 (L/B) WDz (2.21)
  - 圓形建築物：WLz = 3 qh A_z C_L' g_L √(β_L) + R_LR (2.22)
- 扭矩：
  - 簡化式：M_{Tz} = 0.28 B W_D^*(z) (2.23)
  - 詳細式：M_{Tz} = 1.8 q_h C_T' A √(B L) (1/β_T + g_T R_{TR}) (2.24)
- 低層建築物：
  - 順風向：S_{Dz} = 1.49 I^2 V_{10}(C) λ K_{zt}(h) A_z (2.25)
  - 橫風向：S_{Lz} = (0.6 + 0.05 L/B) S_{Dz} (2.29)
  - 扭矩：S_{Tz} = 0.21 B S_{Dz}^* (2.30)

## 4. 批判性審查與疑慮
- 過渡地況的風速垂直分布如何合理分析？
- 鄰近建築物遮蔽效應的折減如何定量？
- 透氣性外牆的風壓折減如何考慮？
- 地形係數 Kzt 計算時條件不滿足的具體處理基準？
- 柔性建築物共振反應因子 R 近似式的適用範圍？
- 固有頻率經驗式 (h<122m) 外的情況？
- 橫風向風力計算式 (2.22) 中各變數定義與單位不明確？
- 扭矩計算式 (2.24) 中 R_{TR} 的內插方法不明確？
- 低層建築物簡化公式是否對所有形狀有效？
